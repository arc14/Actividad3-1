# Actividad-Individual---Control-de-versiones-y-documentaci-n
Este repositorio sera la base para poder realizar Actividad Individual 1 - Control de versiones y documentacion.

En la actividad se han tratado los siguientes apartados:

- Clonado de repositorio
- Manejo de las zonás logicas de la parte local de git
- Uso de comandos add, commit y push
- Manejo de tags y ramas
- Uso de los comandos tag y branch respectivamente
- Comentarios en las distintas clases del proyecto
- Sentencias y exportacion de un documento javaDoc
- Completar el archivo README.md del repositrio


El repositorio contiene el proyecto cedido cuya funcion es la de claculo de areas de algunas formas geometricas.
Tambien inlcuye documento JavaDoc con descripciones sencillas de los diversos apartados del proyecto respecto al lenguaje Java
